# Data Dictionary

This directory contains datasets used for the AI and Economics research project.

## `nigeria_economic_digital_data.csv`
Source: World Bank API (scraped programmatically).
Variables:
- `Year`: The year of the data point.
- `gdp_growth`: Annual GDP Growth (%) for Nigeria (Indicator: `NY.GDP.MKTP.KD.ZG`).
- `financial_inclusion`: Account ownership at a financial institution or with a mobile-money-service provider (% of population ages 15+) (Indicator: `FX.OWN.TOTL.ZS`).
- `broadband_subscriptions`: Fixed broadband subscriptions (per 100 people), serving as a proxy for digital/AI readiness (Indicator: `IT.NET.BBND.P2`).
